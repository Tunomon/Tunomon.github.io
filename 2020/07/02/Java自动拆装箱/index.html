Java自动拆装箱

2020-07-02 • 标签: Java基础

  1. 自动拆装箱

    int变量->Integer变量，装箱

    Interger变量->Int变量，拆箱

    原始类型：byte，short，long，int，boolean，float，double，char

    封装类：Byte，Short，Long，Integer，Boolean，Float，Double，Character

  2. 操作

    自动装箱，是调用valueOf;

    自动拆箱，调用intValue

    缓存范围是-128~127
